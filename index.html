PROJECT 01

ELW/ELS LP 트레이딩 시스템

Java React FastAPI WPF
PROBLEM

→
SOLUTION

초저지연 호가 엔진 + 실시간 리스크 모니터링 + 웹/데스크톱 이중 클라이언트 구축.

→
RESULT

호가 응답 < 1ms. 수백 종목 동시 운영. 실시간 포지션·손익 대시보드.

PROBLEM
